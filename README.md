# Lista de Tarefas

Aplicação de lista de tarefas desenvolvida do zero com HTML5, CSS3 e JavaScript puro, sem frameworks ou bibliotecas externas.

## 🔗 Acesse

👉 [[arthurlaurentis1.github.io/Lista-tarefas.ArthurLaurentis](https://arthurlaurentis1.github.io/Lista-tarefas.ArthurLaurentis)

## 🛠️ Tecnologias utilizadas

- HTML5 semântico
- CSS3 puro — sem Bootstrap, sem Tailwind
- JavaScript puro (Vanilla JS) — sem frameworks ou bibliotecas
- Lógica CRUD Completa: Implementação de um fluxo dinâmico para adicionar, listar, editar e remover tarefas, manipulando arrays e estados na memória do navegador de forma eficiente.
- Tratamento de dados para validação de inputs do usuário. 
- UI/UX fluida com animações CSS e feedbacks visuais instantâneos.
- Manipulação dinâmica do DOM (criação e remoção de elementos em tempo real).
